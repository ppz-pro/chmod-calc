# Chmod 计算器
一个伪实用小工具，用来计算 linux 的 chmod 命令写法，inspired by [this site](https://linuxhandbook.com/chmod-calculator/) and [this site](https://newcss.net/)

[点击进入](https://ppz-pro.github.io/chmod-calc/)

## 特点
+ ✅小，无任何依赖 <6kB
+ ✅支持电脑和移动端
+ ✅自动切换亮/暗模式（跟随系统）
+ ✅中英文切换（用户没设置时，优先使用浏览器默认语言）
+ 可安装为 PWA
